# Scrutiny Disk Dashboard

**Scrutiny** 데이터를 기반으로 한 프리미엄 드라이브 상태 모니터링 대시보드입니다.  
사용자 경험(UX)과 시각적 완성도를 극대화하여, 단순한 모니터링을 넘어선 시스템 관리 경험을 제공합니다.

![Dashboard Preview](assets/127.0.0.1_6091_%20(3).png)

## 📌 주요 특징 (V2)

- **프리미엄 UI/UX**: 글래스모피즘 디자인과 탄성 애니메이션을 적용한 최신 웹 인터페이스.
- **듀얼 모드 지원**: 
  - **리스트 뷰**: 많은 드라이브를 집약적으로 관리하기에 최적화된 레이아웃.
  - **카드 뷰**: 개별 드라이브의 상태를 시각적으로 강조한 그리드 레이아웃.
- **상세 분석 차트 (Analytics)**: 날짜별 누적 쓰기량 추이, 일일 쓰기량 변화, 잔여 수명 변화를 그래프로 시각화.
- **스마트 수명 계산**: NVMe/SATA 드라이브의 SMART 속성을 분석하여 예상 수명 종료일(EOL) 및 예비 영역 상태 표기.
- **사용자 편의성**:
  - **드래그 앤 드롭**: 드라이브 정렬 순서를 자유롭게 변경하고 브라우저에 자동 저장.
  - **실시간 언어 변경**: 한국어(KO)와 영어(EN)를 매끄러운 트랜지션과 함께 즉시 전환.

## 📁 버전 관리 (Versioning)

이 리포지토리는 두 가지 버전을 병행하여 지원합니다:
- **[Latest / V2 (루트)](./)**: 금일 완성된 최신 기능과 고도화된 디자인이 적용된 버전입니다.
- **[Legacy / V1 (v1/)](./v1)**: 안정성이 검증된 이전 세대의 클래식한 인터페이스입니다.

## 🚀 빠른 시작 (Docker)

```yaml
services:
  disk-dashboard:
    image: grt47/disk_info:latest
    container_name: disk-dashboard
    ports:
      - "6091:80"
    environment:
      - SCRUTINY_BASE=http://[YOUR_SCRUTINY_IP]:8080
    restart: unless-stopped
```

## 🛠️ 커스터마징 가이드

본 프로젝트는 `disk_info.php` 단일 파일 내에 모든 로직이 포함되어 있어 수정이 매우 용이합니다.
- **스타일**: CSS 변수(`:root`)를 통해 테마 색상 및 블러 강도 조절 가능.
